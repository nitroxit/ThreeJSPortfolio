## Install & run

Make sure you have nodejs and npm installed ( You will want to use Node 16.8 or lower). Install dependencies with:

```bash
npm i --force
```

Once it's done start up a local server with:

```bash
npm start
```

To create a production build:

```bash
npm run build
```
